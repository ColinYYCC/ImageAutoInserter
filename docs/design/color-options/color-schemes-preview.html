💡 推荐方案

方案 C (融合方案) - 结合了莫兰迪色系的柔和质感和青绿色的活力感。
主色使用青绿色提升视觉冲击力和专业感，背景和辅助色保留莫兰迪的柔和特质，
既保证了工具应用的专业性，又兼顾了长时间使用的舒适度。
