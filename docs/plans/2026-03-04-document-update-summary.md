## ✅ 已完成的更新

### 1. **核心 API 文档更新** (`docs/api-reference.md`)

#### 1.1 图片嵌入方法统一
- **变更**: 统一为智能判断方法 `embed_image()`
- **详情**: 根据参数类型自动判断（字符串=文件路径，bytes=图片数据）
- **影响**: 删除了 `embed_image_from_data()` 方法，简化 API

```python
def embed_image(
    self, 
    row: int, 
    column: str, 
    source: Union[str, bytes],  # 智能判断
    width: int = 180,
    height: int = 138
) -> None:
```

#### 1.2 字段命名优化
- **变更**: `ImageInfo.file_path` → `ImageInfo.source_path`
- **原因**: 更清晰，避免歧义（文件夹=真实路径，压缩包=压缩包内路径）

#### 1.3 初始化参数统一
- **变更**: `ExcelProcessor.__init__(file_path, read_only=False)`
- **原因**: 统一文档说明，支持 read_only 模式节省内存

### 2. **架构文档更新** (`docs/architecture.md`)

#### 2.1 ImageInfo 数据结构
```python
@dataclass
class ImageInfo:
    product_code: str        # 商品编码
    sequence: str            # 序号 (01/02/03)

    image_data: bytes        # 图片二进制数据
    format: str              # 图片格式
    width: int = 0           # 图片宽度（新增）
```

#### 2.2 图片处理器说明

- **新增**: 压缩包内存处理详细说明

- **新增**: 完整的 ZIP 压缩包内存处理代码示例
