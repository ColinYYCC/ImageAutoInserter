## 🔍 DevTools 检查

### 打开 DevTools
- **Chrome/Edge**: F12 或 Cmd+Option+I (Mac)
- **Safari**: Cmd+Option+I (Mac)
- **Firefox**: F12 或 Cmd+Option+I (Mac)

### 检查元素
1. 右键点击卡片 → "检查"
2. 查看 Elements 面板
3. 确认 CSS 类名正确
4. 查看 Styles 面板
5. 确认 CSS 变量应用正确

### 查看 Console
- [ ] 无错误信息
- [ ] 可能有 React 开发模式警告（正常）

### 查看 Network
- [ ] 加载 main.tsx
- [ ] 加载 App.tsx
- [ ] 加载 FilePicker.tsx
- [ ] 加载 global.css
- [ ] 加载 FilePicker.module.css
