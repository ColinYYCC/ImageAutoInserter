# ImageAutoInserter

图片自动插入工具 - 根据商品编码自动将图片嵌入 Excel 表格

## 功能特点

- 自动匹配商品编码，批量插入图片到 Excel
- 支持文件夹、ZIP、RAR、7Z 格式的图片来源
- 实时进度显示
- 自动更新功能

## 下载

请访问 [Releases](https://github.com/ColinYYCC/ImageAutoInserter/releases) 页面下载最新版本。

## 使用说明

1. 选择图片来源（文件夹或压缩包）
2. 选择 Excel 文件
3. 点击"开始处理"
4. 等待处理完成

## 系统要求

- macOS 10.14+ / Windows 10+ / Linux
- Python 3.8+
