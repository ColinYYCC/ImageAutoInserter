# ImageAutoInserter 手动测试清单

> 本清单用于最终验收测试，确保所有功能在真实环境中正常工作。
> **自动化测试**：Web 层测试由 Playwright 执行，本清单补充 Electron 桌面专有功能测试。

---

## 一、测试环境准备

### 1.1 必要条件

| 项目 | 要求 |
|------|------|
| Node.js | >= 18.0.0 |
| Python | >= 3.9 |
| 操作系统 | macOS 11+ / Windows 10+ |
| 测试文件 | 见下方测试夹具目录 |

### 1.2 测试夹具准备

测试夹具文件位于 `tests/e2e/fixtures/`：

| 文件 | 说明 | 生成状态 |
|------|------|----------|
| `test-images.zip` | 有效图片压缩包（包含 SKU001.jpg, SKU002.png） | ✅ 自动生成 |
| `test-data.xlsx` | 有效 Excel（包含"商品编码"列） | ✅ 自动生成 |
| `corrupted.zip` | 损坏的 ZIP 文件 | ✅ 自动生成 |
| `invalid-excel.xlsx` | 无商品编码列的 Excel | ❌ 需手动创建 |
| `empty-folder/` | 空文件夹 | ✅ 自动生成 |
| `large-dataset.zip` | 大文件（100+ 图片） | ❌ 需手动创建 |

### 1.3 应用启动方式

```bash
# 方式 1：开发模式（需要先启动 Vite）
npm run dev

# 方式 2：仅 Electron（使用已构建的 dist）
open dist/mac-arm64/ImageAutoInserter.app

# 方式 3：Windows
start dist\win-unpacked\ImageAutoInserter.exe
```
