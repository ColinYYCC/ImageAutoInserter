# ImageAutoInserter 验收检查清单 (v1.2)

## 核心功能验收

### 图片处理

- [ ] 能够从文件夹读取图片并正确解析商品编码
- [ ] 能够从 ZIP 压缩包提取图片并正确解析商品编码（**完全内存处理，不解压**）
- [ ] 能够从 RAR 压缩包提取图片并正确解析商品编码（**完全内存处理，不解压**）
- [ ] 图片命名格式验证正确（商品编码 - 序号。格式）
- [ ] 序号 01/02/03 正确对应 Picture 1/2/3
- [ ] **图片尺寸固定为 180×138 像素**
- [ ] **保持原画质，不进行有损压缩**
- [ ] ImageInfo 使用 source_path 字段（非 file_path）

### Excel 处理

- [ ] 能够自动识别包含"商品编码"列的 Sheet
- [ ] **能够动态识别表头位置**（表头不一定在第一行）
  - [ ] 遍历表格每一行查找"商品编码"所在行
  - [ ] 正确处理表头前的标题行、空行
  - [ ] 处理合并单元格、不规则格式
- [ ] **能够智能检查并添加 Picture 1/2/3 列**
  - [ ] 检查表头行最后一列是否有文本内容
  - [ ] 如最后一列有文本 → 在其后添加 Picture 1/2/3
  - [ ] 如最后一列为空 → 直接在该行末尾添加 Picture 1/2/3
- [ ] 能够正确检测"Picture 1/2/3"列是否存在
- [ ] **图片能够正确嵌入单元格**（支持文件路径和二进制数据两种方式）
- [ ] **图片嵌入尺寸固定为 180×138 像素**
- [ ] 输出文件命名正确（原文件名_含图.xlsx）
- [ ] 处理多行数据时逐行正确匹配
- [ ] ExcelProcessor 支持 read_only 模式

### 错误处理

- [ ] **错误提示详细化**（包含具体文件、行号、原因）
- [ ] **重试机制正常工作**（最多 3 次）
- [ ] **跳过机制正常工作**（记录错误继续处理）
- [ ] **错误报告生成功能**（生成错误日志文件）
- [ ] 错误提示清晰易懂（非技术术语）

### 进度显示

- [ ] **实时显示当前处理进度**（当前行/总行数）
- [ ] **显示当前处理动作**（"读取 Excel" / "匹配图片" / "嵌入 Picture 1"）
- [ ] **显示当前商品编码**
- [ ] **显示图片来源**（文件夹路径 或 压缩包名称）
- [ ] **预估剩余时间准确**
- [ ] **进度更新频率合理**（每 5 行更新一次，避免卡顿）

### v0.2.0 功能增强验收

#### 错误日志优化

- [ ] **只记录失败行**（不记录成功日志）
- [ ] **智能错误分类**（Excel 公式错误 vs 图片缺失）
- [ ] **精准问题定位**（数据问题 vs 图片文件问题）
- [ ] **操作引导**（为每种错误类型提供解决建议）
- [ ] 验证日志量减少 99.8%

#### Picture 字段变体识别

- [ ] 支持 24 种变体识别（Picture, Photo, Image, Figure, 图片，照片，图像，图等）
- [ ] 单复数形式识别正确
- [ ] 缩写识别正确（Img, Fig.）
- [ ] 拼写纠错正确（Photoes→Photos 等）
- [ ] 大小写统一（首字母大写）

#### 表头保持策略

- [ ] 识别但不修改原始表头
- [ ] 内部标准格式映射正确
- [ ] 新增列使用标准格式（Picture X）

#### 动态列扩展

- [ ] 支持超过 3 张图片（最多 10 个 Picture 列）
- [ ] 按需添加列（只添加实际需要的列数）
- [ ] 智能扫描确定最大图片数
- [ ] 避免添加空白列

#### 智能列位置优化

- [ ] 智能查找第一个空白列
- [ ] 避免覆盖已有数据的列
- [ ] 紧凑布局（靠近数据列）
- [ ] 自动跳过空表头列
