## 功能验收检查清单

### 1. 图片处理功能

#### 1.1 图片读取

- [ ] 能够从文件夹读取图片并正确解析商品编码
- [ ] 能够从 ZIP 压缩包提取图片并正确解析商品编码（完全内存处理，不解压）
- [ ] 能够从 RAR 压缩包提取图片并正确解析商品编码（完全内存处理，不解压）
- [ ] 能够从 7Z 压缩包提取图片并正确解析商品编码（完全内存处理，不解压）

#### 1.2 图片格式

- [ ] 支持 JPG 格式
- [ ] 支持 JPEG 格式
- [ ] 支持 PNG 格式
- [ ] 图片命名格式验证正确（商品编码-序号.格式）
- [ ] 序号 01-10 正确对应 Picture 1-10

#### 1.3 图片质量

- [ ] 图片尺寸固定为 180×138 像素
- [ ] 保持原画质，不进行有损压缩
- [ ] 图片正确嵌入单元格（非链接）

#### 1.4 性能

- [ ] 图片流式加载正常工作
- [ ] 缓存管理正常工作（最大 50MB）
- [ ] 内存使用合理
